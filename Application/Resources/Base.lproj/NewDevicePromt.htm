To add new devices/nodes, you could:

1. Open https://personal.house/ on a new device.
2. Choose the platform corresponding to the new device OS, and then install.
3. Get invitation code from this device (not the new device). (Personal House -> Preferences -> Invite other devices.)
4. On the new device, open Personal House app. if the new device runs on Windows, open "Configure Personal Cloud" from the Start menu.
5. Input the invitation code in the "Welcome" page on the new device. To be successful, the two devices have to be in one intranet (Multicasting enabled)

Now you can visit the others devices’ files from one device.
